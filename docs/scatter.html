The Scatter modules

Overview

The module has two modules:

  • scatter

  • scattervec

They contain the same functionality. The only differ in that scattervec takes a single vec (vector) instead of three floats for x,y,z.

API: scatter

sbs_utils.scatter.arc(count, x, y, z, r, start=0.0, end=90.0, random=False)

Calculate the points along an circular arc

count: int

The number of points to generate

x,y,z: float,float,float

the center point/origin

r:radius start: float=0, optional

the angle to start at in degrees

end: float=360, optional

the angle to end at in degrees
